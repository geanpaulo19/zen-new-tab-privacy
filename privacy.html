Zen Tabs - Privacy Policy

Data Collection

This extension only requests access to your location to display current weather information. No other personal data is collected.

API Usage

The extension fetches data from the following APIs:

  • Unsplash: Retrieves daily background images.
  • Open-Meteo: Retrieves current weather data using your location.
  • Nominatim (OpenStreetMap): Converts coordinates into a readable location name.

No personal information is sent to these APIs beyond your approximate location for weather purposes.

Local Storage

The extension stores minimal settings locally in your browser:

  • 12/24 hour time preference
  • Cached background image and weather data (to reduce API calls)

All stored data remains on your device and is never transmitted elsewhere.

Third-Party Links

Background image credits may link to Unsplash photographer pages.
